Built a Real-Time Polling App with FastAPI & WebSockets 🚀
For one of my Backend Development and DevOps Fundamentals course my mini project, I developed a realtime polling application where users can vote and instantly see updates across multiple browser tabs without refreshing the page.
The project includes:
REST API endpoints with FastAPI
WebSocket-based realtime communication
Live vote broadcasting to connected clients
Dockerized setup using Docker Compose
While building this project, I gained more practical experience with realtime backend systems, WebSocket connections, and client-server communication.
I also containerized the application with Docker to make the setup and deployment process easier.
